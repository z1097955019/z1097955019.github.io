1 备份

  1. 修改为两个分支：master和hexo；设置hexo为默认分支；
  2. 将刚刚创建的新仓库clone至本地，将之前的hexo文件夹中的_config.yml，themes/，source/，scaffolds/，package.json，.gitignore复制至WincerChan.github.io文件夹；
  3. 将themes/next/(我用的是NexT主题)中的.git/删除，否则无法将主题文件夹push；
  4. 在WincerChan.github.io文件夹执行npm install和npm install hexo-deployer-git（这里可以看一看分支是不是显示为hexo）；
